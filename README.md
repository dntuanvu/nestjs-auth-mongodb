## Running the app

```bash
# development
$ npm start

# watch mode
$ npm run start:dev
```

```bash
# docker (to have a ready running application with database attached)
docker-compose build

docker-compose up
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
